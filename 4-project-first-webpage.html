👥 前回のおさらい：HTML/CSSの基礎

🔍 HTML/CSSの役割の復習

HTML（エイチティーエムエル）とは「HyperText Markup Language（ハイパーテキスト・マークアップ・ランゲージ）」の略で、Webページの構造を定義する言語です。これはちょうど家の骨組みや間取りのようなものです。

CSS（シーエスエス）とは「Cascading Style Sheets（カスケーディング・スタイル・シート）」の略で、HTMLで作った構造の見た目をデザインする言語です。これは家の壁紙や家具、照明など内装に相当します。

📌 これまでに学んだHTMLタグ
  • <h1> - 大見出し（Heading 1）
  • <p> - 段落（Paragraph）
  • <strong> - 強調（Strong）
🎨 これまでに学んだCSSプロパティ
  • color - 文字の色
  • font-size - 文字の大きさ
  • background-color - 背景の色

📋 シンプルな自己紹介ページの作成

🔍 HTMLコーディングの基本手順

HTMLコードはタグ（Tag：指示書）を使って書きます。タグは「<」と「>」で囲まれた指示で、Webブラウザに「この部分はこういう役割です」と教えます。

📌 基本的なページを作るHTMLコード
<h1>太郎のページ</h1>

<p>はじめまして！</p>
<p>僕の名前は太郎です。</p>
<p>中学2年生です。</p>

<p><strong>よろしくお願いします！</strong></p>
🎨 基本的なページのデザインを決めるCSSコード
<style>
h1 {
    color: green;
    font-size: 36px;
}

p {
    color: darkgreen;
    font-size: 18px;
}

strong {
    color: orange;
    font-size: 20px;
}
</style>

💡 コードの解説
  • <h1> - 見出しタグ（Heading）：ページのタイトルや大見出しを表示します。数字は1〜6まであり、小さくなっていきます。
  • <p> - 段落タグ（Paragraph）：文章の段落を表します。自動的に上下に余白ができます。
  • <strong> - 強調タグ：内容を強調して太字で表示します。
  • color: green; - 文字の色を緑色に設定します。
  • font-size: 36px; - 文字の大きさを36ピクセル（px）に設定します。

📝 リスト機能でコンテンツを整理する

🔍 リストとは

リスト（List）は、複数の項目を整理して表示するための機能です。買い物リストや手順書のように、関連する項目を箇条書きで表示できます。

📌 リストを作るHTMLタグ
<ul>        ← リスト全体の始まり（Unordered List：順序なしリスト）
    <li>サッカー</li>     ← 1つ目の項目（List Item：リスト項目）
    <li>ゲーム</li>       ← 2つ目の項目
    <li>読書</li>         ← 3つ目の項目
</ul>       ← リスト全体の終わり

💡 リストタグの解説
  • <h2> - 中見出しタグ：<h1>より小さい見出しで、セクションのタイトルに使います。
  • <ul> - 順序なしリストタグ（Unordered List）：箇条書きのリストを作成します。自動的に各項目の前に黒丸（・）が表示されます。
  • <li> - リスト項目タグ（List Item）：リストの各項目を定義します。
⚠️ 注意点

<ul>と<li>タグは必ずセットで使います。<ul>の中に<li>を入れる構造です。<li>だけでは正しく表示されません。

📷 画像を追加してページを視覚的に強化する

🔍 画像タグとは

画像タグ（Image Tag）は、Webページに写真やイラストなどの画像を表示するためのHTMLタグです。視覚的な情報を追加することで、ページの魅力が大きく向上します。

📌 画像を追加するHTMLタグ
<img src="写真の場所" alt="写真の説明">

※ <img>タグは、他のタグと違って終了タグ（</img>）がありません。これを「自己完結型タグ」といいます。

💡 imgタグの属性（アトリビュート）
  • src - ソース属性（Source）：画像ファイルの場所（URL）を指定します。
  • alt - 代替テキスト属性（Alternative Text）：画像が表示できない場合や、スクリーンリーダー（目の不自由な方が使う読み上げソフト）のために、画像の内容を説明するテキストを設定します。
📌 画像を追加した完成版HTMLコード
<h1>太郎のページ</h1>

<img src="https://via.placeholder.com/200x200/90EE90/000000?text=太郎" alt="太郎くんの写真">
