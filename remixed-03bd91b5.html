CSSの魔法世界: 見た目を変える魅力的な技術

1. 前回のおさらい - HTMLによる構造づくり

私たちは前回、HTMLタグを使ってWebページの「骨組み」を作る方法を学びました。HTMLはHyperText Markup Language（ハイパーテキスト・マークアップ・ランゲージ）の略で、Webページの構造を定義する言語です。

前回学んだHTMLの例：

<h1>夏祭りの思い出</h1>
<p>昨日、家族で地元の夏祭りに行きました。</p>
<p>綿あめが<strong>とても美味しかった</strong>です。</p>

しかし、HTMLだけでは文字や背景の色、大きさなどの「見た目」を自由に変えることができません。白い背景に黒い文字だけでは、ちょっと寂しい感じがしますね。

2. CSSって何だろう？ - お人形の着せ替え例え

CSSは「Cascading Style Sheets（カスケーディング・スタイル・シート）」の略で、Webページの見た目を装飾するための言語です。「カスケーディング」とは「滝のように上から下へ流れる」という意味で、スタイルが適用される仕組みを表しています。

CSS（Cascading Style Sheets）: Webページの見た目を決めるための言語。色、大きさ、配置など、ページの視覚的な部分をすべて制御します。

CSSをわかりやすく例えると「お人形の着せ替え」のようなものです。HTMLはお人形の体（構造）、CSSはお人形に着せる洋服（見た目）と考えるとイメージしやすいでしょう。

HTML（お人形の体）
CSS赤ドレス
CSS青ドレス

同じお人形（HTML）でも、着せる洋服（CSS）によって印象がまったく変わります。Webページも同じで、HTMLの内容はそのままでも、CSSを変えるだけで見た目を大きく変えることができるのです。

4. CSSの魔法ファイルを作る

CSSを適用するには、大きく分けて2つの方法があります。今回は簡単な方法として、HTMLファイルの中に直接CSSを書く「インラインスタイル」を使ってみましょう。

4.1 styleタグを使った魔法のコード

HTMLファイルの中に、<style>タグを使ってCSSコードを書きます。このタグは「ここからCSSの魔法が始まりますよ」という印になります。

HTMLファイルにCSSを追加する例：

<h1>夏祭りの思い出</h1>
<p>昨日、家族で地元の夏祭りに行きました。</p>
<p>綿あめが<strong>とても美味しかった</strong>です。</p>

<style>
h1 {
    color: blue;
    font-size: 36px;
}

p {
    color: gray;
    font-size: 18px;
}

strong {
    color: red;
}

body {
    background-color: lightyellow;
}
</style>

専門用語：インラインスタイル - HTMLファイル内に直接CSSを書く方法のこと。小規模なページや学習段階では便利ですが、大きなサイトでは通常、別ファイルにCSSを書きます。

4.2 外部CSSファイルを作る方法

実際のWebサイト制作では、CSSを別ファイルとして保存することが一般的です。これを「外部スタイルシート」と呼びます。

外部CSSファイルの作り方：

1. テキストエディタで新しいファイルを作成

2. CSSコードを入力

h1 {
    color: blue;
    font-size: 36px;
}

p {
    color: gray;
    font-size: 18px;
}

strong {
    color: red;
}

body {
    background-color: lightyellow;
}

3. 「style.css」という名前で保存

4. HTMLファイルから読み込む

<head>
    <link rel="stylesheet" href="style.css">
</head>

専門用語：外部スタイルシート - CSSを別ファイルとして保存し、HTMLから参照する方法。複数のページで同じデザインを共有したい場合に便利です。

5. CSSの基本ルール - 魔法の構造

CSSは「誰に」「何を」「どうする」という3つの要素で構成されています。これを専門的には「セレクタ」と「宣言ブロック」と呼びます。

h1 { color: blue; }

5.1 セレクタ - 誰に魔法をかけるか

セレクタ（Selector）は「誰に魔法をかけるか」を指定します。例えば、h1はすべての見出し（h1タグ）に、pはすべての段落（pタグ）に魔法をかけることを意味します。

セレクタ（Selector）: CSSで装飾したいHTML要素を指定するための記述。HTMLタグ名、クラス名、ID名などで指定できます。

5.2 プロパティと値 - どんな魔法をかけるか

宣言ブロック内のプロパティ（Property）と値（Value）のペアで「どんな魔法をかけるか」を指定します。

プロパティ（Property）: 変更したいスタイルの種類を指定する（例: color, font-size, background-color）

値（Value）: プロパティに対して設定する具体的な値（例: blue, 36px, lightyellow）

5.3 重要なCSSプロパティと値

プロパティ 説明 値の例
color 文字の色を指定 red, blue, #FF0000
font-size 文字の大きさを指定 12px, 2em, 150%
background-color 背景色を指定 yellow, #FFFF00, lightyellow
font-weight 文字の太さを指定 bold, normal, 700
text-align 文字の配置を指定 left, center, right

8. CSSの適用プロセス - 魔法がかかる仕組み

CSSがどのようにHTMLに適用されるのか、そのプロセスを図解してみましょう。

CSSの適用プロセス
flowchart TD
    A[ブラウザがHTMLを読み込む] --> B[HTMLの構造を解析]
    B --> C[CSSを読み込む]
    C --> D[CSSのルールを解析]
    D --> E[HTMLの各要素にスタイルを適用]
    E --> F[最終的なページを表示]
    
    subgraph CSS適用順序
    G[ブラウザのデフォルトスタイル]
    H[外部スタイルシート]
    I[内部スタイル]
    J[インラインスタイル]
    end
    
    G --> H --> I --> J

専門用語：カスケード（Cascade） - 複数のスタイルが競合した場合に、どのスタイルが優先されるかを決める仕組み。通常、より具体的なセレクタほど優先度が高くなります。
